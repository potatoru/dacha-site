Кланы на сервере несут чисто социальную функцию — **клановый префикс** в чате и отдельный **клан-чат**. Клан-вары, 
захват территорий, банк и т.д. отсутствуют.

Для создания клана вам необходима [активная подписка](/donate). Для участия в клане подписка не нужна.

### Команды
- `/clan create <имя_клана>` — создать клан.
- `/clan tag <тег>` — добавить клану тег, который будет отображаться в чате и табе. Ограничение — **5 символов**.
- `/clan invite <ник>` — пригласить игрока.
- `/clan accept` — принять приглашение.
- `/clan deny` — отклонить приглашение.
- `/clan leave` — выйти из клана.
- `/clan kick <ник>` — выгнать игрока.
- `/clan rank <ник> <ранг>` — изменить ранг игрока.
- `/clan desc <описание>` — изменить описание клана.
- `/clan rename <название>` — изменить название клана.

Для того, чтобы удалить клан, вам необходимо просто выйти из клана. 
При этом вы должны быть единственным лидером этого клана.

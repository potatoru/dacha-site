На сервере установлен плагин, позволяющий приобретать головы для декора. 
Вы можете использовать их, например, для украшения жилища.

Стоимость одной головы — **3 алмаза**.

### Команды
- `/hdb` — открыть каталог голов.
- `/hdb search <запрос>` — поиск по каталогу.

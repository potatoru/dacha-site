### Команды
- `/ps add <ник>` — добавить игрока в приват.
- `/ps addowner <ник>` — добавить игрока как владельца.
- `/ps remove <ник>` — удалить игрока.
- `/ps removeowner <ник>` — удалить владельца.
- `/ps flag` — настройка флагов.
- `/ps hide` — скрыть блок привата.
- `/ps unhide` — показать блок привата.
- `/ps unclaim` — убрать приват и вернуть блок.

Перед тем, как начать работать с приватами, ознакомьтесь с [правилами](/rules) привата территории.
